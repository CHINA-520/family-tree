# 📜 极简单文件族谱系统 (变量化适配版)

基于 **Cloudflare Workers + KV** 构建的轻量级数字化族谱，专为家族传承设计。

## ✨ 核心亮点
- **一键改姓**：通过文件顶部的 `SYS_CONF` 配置，秒变任意家族专属系统。
- **智能搜索**：内置实时搜索框，支持按姓名、地点、简介快速定位成员。
- **四维视图**：
  - **卡片视图**：精美卡片式排版，适合手机阅读详细生平。
  - **树状视图**：经典的垂直宗谱脉络，一目了然。
  - **成员列表**：标准表格视图，方便快速核对与打印。
  - **关系图表**：基于 D3.js 的交互式关系图，支持缩放、拖拽。
- **管理便捷**：支持一键录入后代、自动计算代数、生卒年动态展示。
- **双重加密**：访问需“查看密码”，修改需“管理员密码”。



---

## 🚀 快速部署指引

1. **创建数据库**：在 Cloudflare 控制台新建一个 KV 命名空间（建议起名 `MY_NOTES_KV`）。
2. **复制代码**：将本仓库的 `index.js` 内容复制到 Worker 脚本编辑器中。
3. **个性化配置**：修改脚本最上方的 `SYS_CONF` 对象（姓氏、颜色、密码）。
4. **绑定变量**：在 Worker 的 **Settings -> Variables** 中添加绑定：
   - Variable name: `MY_NOTES`
   - KV namespace: 选择你刚才创建的空间。
5. **部署生效**：点击保存并部署。

## 🛠 使用贴士
- **录入顺序**：先录入“始祖”，然后点击其卡片使用“录入后代”功能，系统会自动建立父子连接。
- **打印指南**：在电脑端进入“成员列表”或“卡片视图”，直接使用浏览器打印（Ctrl+P）即可生成纸质存档。
- **定期备份**：在“数据管理”页面定期导出 JSON 备份，确保万无一失。
